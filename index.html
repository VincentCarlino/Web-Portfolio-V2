Startpage

Startpages are HTML files that typically list your most frequently used sites and services. It's common practice to simply run them locally as your homepage for your web browser. I constructed my own startpage with functional dropdown menus and a base color switcher using native JavaScript in early 2017. This project acted as my introduction to web-design and HTML/CSS and JavaScript. Since then, I've continued to improve my startpage so it now includes more polished animations and visuals.
